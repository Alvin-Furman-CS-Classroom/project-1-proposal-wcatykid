# Task 1: Simple Python Script

## Overview

In this task, you will learn how to use an AI agent to create a simple Python script.

## Steps

### 1. Open your agent chat window or terminal

### 2. Draft your prompt:

> I want to create a simple Python script that prints "Hello, World!"

### 3. Carefully read the output and ask for clarifications as needed

You can (and should!) go back-and-forth with your agent to get a quality result.

### 4. Follow the instructions in the output to create the script

5. Run the script and verify it works
