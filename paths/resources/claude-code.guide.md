# Claude Code Guide

This document describes the Claude Code IDE and how to use it to code with AI agents.

## Overview

Claude Code is a TUI (Terminal User Interface) based agent that can work along side with you no matter what editor you are using. Claude Code and Opus 4.5 in particular is a very powerful combination.
